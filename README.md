knowlr
=======

generic object storage interface

Getting Started
---------------

1. You must have node > 0.8 installed

2. You must have mongodb installed.

3. Install npm dependencies

  ```
  npm install --dev
  ```

5. Populate your database with initial values:

  ```
  cake seed
  ````

5. Start the server:

  ```
  cake watch
  ````

6. Log in as admin:

  ```
  username: admin
  password: admin
  ```
